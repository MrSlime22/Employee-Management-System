Evaluation Result

List of Usability problems found:

Heuristic Violated: Consistency and Standards

Description: Buttons have the same design but different colors, which can confuse users about their functions.

Impact: Users may mistakenly assume that differently colored buttons perform different actions, leading to errors.

Heuristic Violated: User Control and Freedom

Description: There’s no option to undo actions, such as deleting an entry.

Impact: This can make users nervous about making changes since they can’t easily go back if they make a mistake.

Heuristic Violated: Match Between System and the Real World

Description: Some of the terms used in the system are confusing and not what users might expect.

Impact: If users don’t understand the language, it makes it harder for them to use the system effectively and can slow them down.

Heuristic Violated: Aesthetic and Minimalist Design

Description: The interface might have a lot of unnecessary information, which can be distracting.

Impact: A cluttered screen can overwhelm users, making it difficult for them to focus on what they need to do.

Heuristic Violated: Visibility of System Status

Description: The system doesn’t clearly show what’s happening after users take an action, like adding or deleting something.

Impact: This can make users unsure if their action worked, leading to confusion and frustration.

Heuristic Violated: Help Users Recognize, Diagnose, and Recover from Errors

Description: Error messages are often unclear and use complicated language.

Impact: If users can’t understand the error messages, they might struggle to fix problems, leading to frustration.

List of Tasks for Evaluation


1. Log In to the System

- Access the Employee Management System with valid credentials.


2. Add a New Designation

- Navigate to the Designation section and successfully add a new designation.


3. Edit an Existing Designation

- Select an existing designation and modify its details.


4. Delete a Designation

- Remove a designation from the list and confirm the action.


5. Search for a Designation

- Use the search function to find a specific designation by name.


6. View All Departments

- Navigate to the Department section and review the list of all departments.


7. Add a New Department

- Successfully add a new department to the system.


8. Export Employee Data

- Use the export function to download employee data in the appropriate format.


9. View Employee Details

- Select an employee from the list to view their detailed information.


10. Navigate Between Pages

- Use pagination to navigate through multiple pages of employee or department entries.



Conclusion of the Evaluation


The evaluation of the Employee Management System revealed several usability issues that hinder user experience. Key violations of usability heuristics included inconsistencies in button design, lack of feedback after actions, and insufficient error prevention measures. By addressing these issues, such as ensuring consistent design across buttons and providing clear feedback for user actions, the system can significantly enhance usability. Overall, improvements in these areas would lead to a more intuitive and user-friendly experience for employees managing designations and departments within the system.
